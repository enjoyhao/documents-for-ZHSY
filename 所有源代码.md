### 目录结构

```
├─api    // 对后端api服务调用的封装
├─components  // 可复用的组件
│  ├─card
│  └─mini-tab
│
├─libs  // 公用的js库，主用作用为引入对asycn/await的支持
├─mixins // 可复用的小程序组件行为（Behaviors）
├─pages  // 各个页面
│  ├─designQuestionaire
│  ├─fillInQuestionaire
│  ├─index
│  ├─login
│  ├─logs
│  ├─mine
│  ├─newtask
│  ├─profile
│  ├─submitSuccess
│  └─taskDetail
├─static   // 静态资源
│  ├─icons
│  └─imgs
├─store   // 全局状态管理模块
└─utils  // 通用模块
```
